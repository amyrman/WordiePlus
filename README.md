
## Description

Wordie+ is a word guessing web app made for a JavaScript fullstack assignment.

## Installation

### After cloning repo, in project root
1. Install dependencies:
    ```bash
    npm install
    ```
2. Run tests:
    ```bash
    npm test
    ```
3. Start frontend + backend server:
    ```bash
    npm start
    ```
4. Open `localhost:5080` in browser.
