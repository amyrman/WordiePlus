
## Description

Wordie+ is a word guessing web app based on the Wordle concept, made for a JavaScript fullstack assignment.

## Installation

### After cloning repo, in project root
1. Install dependencies:
    ```bash
    npm install
    ```
2. Run tests:
    ```bash
    npm test
    ```
3. Start frontend + backend server:
    ```bash
    npm start
    ```
4. Open in browser:
    ```bash
    localhost:5080
    ```
